# I2C addresses

product-level I2C device addresses.

## addresses

| component | address | config macro        | function |
| --------- | ------- | ------------------- | -------- |
| mp2667    | `0x09`  | `MP2667_I2C_ADDR`   | charger  |
| veml3328  | `0x10`  | `VEML3328_I2C_ADDR` | light    |
| bma253    | `0x19`  | `BMA253_I2C_ADDR`   | accel    |
| bq27421   | `0x55`  | `BQ27421_I2C_ADDR`  | gauge    |
